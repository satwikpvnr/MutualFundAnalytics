\# Mutual Fund Analytics Data Dictionary



\## clean\_nav.csv

\- amfi\_code : Unique AMFI scheme code

\- date : NAV date

\- nav : Net Asset Value



\## clean\_transactions.csv

\- investor\_id : Unique investor identifier

\- transaction\_date : Transaction date

\- amfi\_code : Mutual fund scheme code

\- transaction\_type : SIP / Lumpsum / Redemption

\- amount\_inr : Transaction amount

\- state : Investor state

\- city : Investor city

\- kyc\_status : Verified / Pending



\## clean\_performance.csv

\- amfi\_code : Scheme code

\- return\_1yr\_pct : 1 year return %

\- return\_3yr\_pct : 3 year return %

\- return\_5yr\_pct : 5 year return %

\- sharpe\_ratio : Risk-adjusted return metric

\- expense\_ratio\_pct : Expense ratio %
